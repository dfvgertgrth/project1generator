Графическое Java-приложение для создания школьного/университетского расписания на неделю.

⚡ Основные функции:
- Составление расписания на 5 дней (Пн-Пт) × 7 уроков
- Назначение занятий с выбором: день, урок, предмет, преподаватель
- Быстрое управление через горячие клавиши
- Очистка расписания с подтверждением
- Удобный русскоязычный интерфейс
